# finance-sql-analytics-project
# Finance & Revenue Analytics Project

## Overview
This project simulates a real-world finance reporting system using SQL. It models customer transactions, product sales, and revenue tracking.

## Key Skills Demonstrated
- SQL JOINs (INNER JOIN)
- Aggregations (SUM, COUNT)
- GROUP BY and HAVING
- Window functions (RANK, ROW_NUMBER conceptually)
- Business KPI analysis

## Business Insights Generated
- Customer revenue performance
- Daily sales trends
- High-value customer identification
- Customer segmentation

## Purpose
Built to demonstrate practical SQL skills relevant to Finance Analyst and Business Analyst roles.
